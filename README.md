## Setup Instructions:

Clone the repository: 
```
git clone https://github.com/yourusername/ngo-website.git
```
Navigate to the project directory: 
```
cd ngo-website
```
Install dependencies: 
```
npm install
```
Run the application: 
```
npm start
```
Ensure PHP is set up for backend integration.
